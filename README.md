# Objective

Instacard is an online store that operates an app. Instacart already has very good sales, but they want to uncover more information about their sales patterns. The objective of this study is to analyse customer  behaviour and uncover patterns for growth.

# Data

The dataset comprises information on the following parameters:

- orders
- products
- departments
